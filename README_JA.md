> HoneySelect2 / Koikatsu の不足MOD精密補完ツール

カードに不足MODが表示されて困っていませんか？BetterRepackを手動で検索するのはもう終わりにしましょう。

**ModResolver**はカードの不足MODを自動的に特定し、ワンクリックで精密に補完します。

---

## ✨ 主な特徴

### 🎯 100%精密マッチング — 推測はもう不要
HTTP Range Request技術により、リモートzipmodからmanifest.xml（数KB）のみを取得し、MODのGUIDを直接読み取ります — **ファイル名の曖昧マッチングではなく、MOD識別子の完全一致**で精度100%。キャラクターカード、服装カード、シーンカードからMOD依存関係を自動解析し、ローカルの既存MODと比較して、**本当に不足しているものだけをダウンロード**します。

### ⚡ 超高速インデックス構築 — 1万MODを数分で処理
zipmod全体をダウンロードする必要はありません。manifest.xmlのキー情報のみを抽出してインデックスを構築。200スレッドの同時処理で、12,000以上のMODを数分で完了。増分インデックスにより、新規・変更されたMODのみを処理します。

### 📥 ワンクリック補完 — 簡単＆安全
不足MODを選択して一括ダウンロード。ダウンロードは隔離ディレクトリに保存されるため、元のmodsフォルダは汚れません。

### 🖥️ モダンUI — 直感的＆クリーン
ダークテーマUI、カードカテゴリブラウジング、不足MOD TOP10ランキング、リアルタイムダウンロード進捗 — すべてが手元に。
