还在为卡片显示缺失模组而烦恼？还在手动去 BetterRepack 搜索下载？

**ModResolver** 自动识别卡片缺失的模组，一键精准补全，告别无效下载。

---

## ✨ 核心特色

### 🎯 100% 精准匹配，不再盲猜
基于 HTTP Range Request 技术，仅获取远程 zipmod 中的 manifest.xml（几KB），直接读取模组 GUID —— **不是模糊匹配文件名，而是精确匹配模组标识**，准确率 100%。自动解析角色卡、服装卡、场景卡中的模组依赖，与本地已有模组对比，**只下载你真正缺失的**，不浪费一分带宽。

### ⚡ 极速索引，万级模组分钟级完成
无需下载完整 zipmod，仅提取 manifest.xml 中的关键信息即可建立索引。200 线程并发构建，12000+ 模组几分钟完成。增量索引机制，只处理新增和变更的模组。

### 📥 一键补全，省时省心
检测到缺失模组后，勾选即可批量下载。下载内容隔离存放，不污染原始 mods 目录。

### 🖥️ 现代界面，操作直观
深色主题 UI，卡片分类浏览、缺失 TOP10 排行、实时下载进度，一切尽在掌控。
